HRMS : Adding New Elements

Created by shehab ahmed khalil, last modified on Jun 07, 2023
Adding new page

  • In the pages directory, create a new folder with the name of the page (capitalized) and create a JS file named index.js

    .

  • Inside the JS file, create a component with the name of the page and export it as default

  • Inside App.js, based on the page (requires authentication or not) either add a route directly inside the Routes wrapper (case no authentication), or nest it inside the PrivateRoute Route 

          <Routes>
        <Route path="/login" element={<Login />} />
        <Route element={<PrivateRoute />}>
          <Route path="/" element={<Home />} />
          <Route
            path="/dashboard"
            element={<PrivateRoute element={<Dashboard />} />}
          />
        </Route>
      </Routes>
Adding a shared component (only components that are shared with different pages are considered shared components)

  • In the Components directory, create a new folder with the name of the Component (capitalized) and create a JS file named index.js

  • Inside JS file right down the component with the same name as the file name and export it as default.

  • The component should be documented as the following

    /**
 * @param {Array} prop1
 * @param {Boolean} prop2
 * @returns
 */
const NewComponent = ({ prop1, prop2 }) => {};
export default NewComponent;
